Бинарники Хьюго доступны для всех платформ на&nbsp;[Гитхабе](https://github.com/gohugoio/hugo/releases).  Достаточно скачать нужный архив, распаковать его и&nbsp;работать c&nbsp;программой через терминал.

<br>

## Винда

1. Скачать и&nbsp;распаковать архив в&nbsp;```C:\Hugo\bin```.
2. Запустить командную строку через поиск и&nbsp;выполнить команду:
```sh
cd C:\Hugo\bin
```
Теперь можно работать с&nbsp;Хьюго, но&nbsp;только в&nbsp;этой директории. Нужно добавить путь до&nbsp;Хьюго в&nbsp;переменные окружения.

3. Открыть через поиск «Изменение системных переменных среды», открыть переменные среды, добавить переменную с&nbsp;путем до&nbsp;программы.

Хьюго готов к&nbsp;работе из&nbsp;любого места в&nbsp;системе.

<br>

## Линукс

Хьюго доступен в&nbsp;репозиториях, поэтому можно установить его одной командой и&nbsp;начать пользоваться. Если версия из&nbsp;репозитория не&nbsp;устраивает, можно скачать бинарник и&nbsp;распакавать архив в&nbsp;```~/hugo```.

Программу можно запустить командой:
```console
~/hugo/hugo
```

Это не&nbsp;очень удобно, хотелось&nbsp;бы сократить до&nbsp;простого ```hugo```, чтобы было похоже на&nbsp;остальные команды. За&nbsp;это отвечает переменная окружения ```$PATH```. Вносить изменения в&nbsp;нее или в&nbsp;директории, которые она хранит, сложнее, чем выполнение этой команды:

```console
echo 'export PATH=$PATH:~/hugo' >> ~/.bashrc
```

Теперь при каждом запуске терминала к&nbsp;переменной добавляется путь до&nbsp;Хьюго.

Примечание: если в&nbsp;системе по&nbsp;умолчанию стоит не&nbsp;Bash, а&nbsp;Zsh, то&nbsp;вместо ```.bashrc``` нужно вписать ```.zshrc```. Проверить оболочку: ```echo $SHELL```.

<br>

## Мак

Тут то&nbsp;же самое, что и&nbsp;на&nbsp;Линуксе. Стоит уточнить, что на&nbsp;последних маках по&nbsp;умолчанию стоит Zsh, но&nbsp;вместо ```.zshrc``` рекоммендуется использовать ```.zprofile```.

Либо можно установить пакетный менеджер [Хоумбрю](https://brew.sh/) и&nbsp;через него скачать Хьюго: 
```console
brew install hugo 
```
