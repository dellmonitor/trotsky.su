Хьюго строит сайт из шаблонов. Он применяет заготовленные пользователем правила для генерации сайта. Оформление поста описано одним файлом. В этом файле могут быть ссылки на другие шаблоны: общие для всех страниц подвал (футер), меню навигации, кнопки. Существуют сотни готовых [тем оформления](https://themes.gohugo.io/) Хьюго.

На данном сайте используется тема Тейл. Для ее установки нужно скачать тему с [Гитхаба](https://github.com/EmielH/tale-hugo) и распаковать в папке /themes/tale. Либо подключить тему в виде [подмодуля](https://git-scm.com/book/ru/v2/Инструменты-Git-Подмодули):
```console
git submodule add https://github.com/EmielH/tale-hugo.git themes/tale
```
В конфиге сайта следует указать тему:
```console
echo "theme = 'tale'" >> config.toml
```
