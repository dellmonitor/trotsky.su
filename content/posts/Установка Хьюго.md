Бинарники Хьюго доступны для всех платформ на [Гитхабе](https://github.com/gohugoio/hugo/releases).  Достаточно скачать нужный архив, распаковать его и работать c ним через терминал.

# Винда

1. Нужно скачать и распаковать архив.
2. Запустить командную строку через поиск и выполнить команду:
```console
C:\Users\mayak>cd <\путь\до\папки>
```
Теперь можно работать с Хьюго, но только в этой папке. Нужно добавить путь до Хьюго в переменные окружения.

3. Открыть через поиск «Изменение системных переменных среды», открыть переменные среды, добавить переменную с путем до программы.

Хьюго готов к работе из любого места в системе.

# Линукс

Хьюго доступен репозиториях, поэтому можно установить его одной командой и начать пользоваться. Если версия из репозитория не устраивает, можно скачать бинарник.

1. Скачать и распокавать архив.
2. Открыть папку через терминал.
```console
mayak@alex:~$ cd </путь/до/папки>
```

Программа готова к работе. Чтобы запускать программу из любой директории, нужно в файл из домашнего каталога .bashrc добавить команду  export PATH=$PATH:</путь/до/Хьюго>. Эта команда при каждом запуске терминала указывает информацию, где находится исполняемый файл программы.
```console
mayak@alex:~$ echo 'export PATH=$PATH:/home/damn/Desktop/hugo' >> ~/.bashrc
```
# Мак

Можно скачать бинарник и выполнить первые два пункта для Линукса.

Либо установить пакетный менеджер [Хоумбрю](https://brew.sh/) и через него скачать Хьюго: 
```console
mayak@alex:~$ brew install hugo 
```
