Бинарники Хьюго доступны для всех платформ на [Гитхабе](https://github.com/gohugoio/hugo/releases).  Достаточно скачать нужный архив, распаковать его и работать c ним через терминал.

# Винда

1. Cкачать и распаковать архив в C:\Hugo\bin.
2. Запустить командную строку через поиск и выполнить команду:
```sh
cd C:\Hugo\bin
```
Теперь можно работать с Хьюго, но только в этой директории. Нужно добавить путь до Хьюго в переменные окружения.

3. Открыть через поиск «Изменение системных переменных среды», открыть переменные среды, добавить переменную с путем до программы.

Хьюго готов к работе из любого места в системе.

# Линукс

Хьюго доступен репозиториях, поэтому можно установить его одной командой и начать пользоваться. Если версия из репозитория не устраивает, можно скачать бинарник.

1. Скачать и распокавать архив в ~/hugo.
2. Открыть директорию через терминал:
```console
cd ~/hugo
```

Программа готова к работе. Чтобы запускать программу из любой директории, нужно в файл из домашнего каталога .bashrc добавить команду  export PATH=$PATH:</путь/до/директории/с/программой>. Эта команда при каждом запуске терминала указывает информацию, где находится исполняемый файл программы.
```console
echo 'export PATH=$PATH:~/hugo' >> ~/.bashrc
```
# Мак

Можно выполнить первые два пункта для Линукса.

Либо установить пакетный менеджер [Хоумбрю](https://brew.sh/) и через него скачать Хьюго: 
```console
brew install hugo 
```
