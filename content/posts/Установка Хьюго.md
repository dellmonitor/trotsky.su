Бинарники Хьюго доступны для всех платформ на&nbsp;[Гитхабе](https://github.com/gohugoio/hugo/releases).  Достаточно скачать нужный архив, распаковать его и&nbsp;работать c&nbsp;программой через терминал.

# Винда

1. Скачать и&nbsp;распаковать архив в&nbsp;```C:\Hugo\bin```.
2. Запустить командную строку через поиск и&nbsp;выполнить команду:
```sh
cd C:\Hugo\bin
```
Теперь можно работать с&nbsp;Хьюго, но&nbsp;только в&nbsp;этой директории. Нужно добавить путь до&nbsp;Хьюго в&nbsp;переменные окружения.

3. Открыть через поиск «Изменение системных переменных среды», открыть переменные среды, добавить переменную с&nbsp;путем до&nbsp;программы.

Хьюго готов к&nbsp;работе из&nbsp;любого места в&nbsp;системе.

# Линукс

Хьюго доступен репозиториях, поэтому можно установить его одной командой и&nbsp;начать пользоваться. Если версия из&nbsp;репозитория не&nbsp;устраивает, можно скачать бинарник.

1. Скачать и&nbsp;распакавать архив в&nbsp;```~/hugo```.
2. Открыть директорию через терминал:
```console
cd ~/hugo
```

Программа готова к&nbsp;работе. Чтобы запускать программу из&nbsp;любой директории, нужно в&nbsp;файл из&nbsp;домашнего каталога .bashrc добавить команду  ```export PATH=$PATH:</путь/до/директории/с/программой>```. Эта команда при&nbsp;каждом запуске терминала указывает информацию, где&nbsp;находится исполняемый файл программы.
```console
echo 'export PATH=$PATH:~/hugo' >> ~/.bashrc
```
# Мак

Можно выполнить первые два пункта для Линукса.

Либо установить пакетный менеджер [Хоумбрю](https://brew.sh/) и&nbsp; через него скачать Хьюго: 
```console
brew install hugo 
```
