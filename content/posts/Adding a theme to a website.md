Хьюго строит сайт из&nbsp;шаблонов. Он&nbsp;применяет заготовленные пользователем правила для генерации сайта. Оформление поста описано одним файлом. В&nbsp;этом файле могут быть ссылки на&nbsp;другие шаблоны: общие для всех страниц подвал (футер), меню навигации, кнопки. Существуют сотни готовых [тем оформления](https://themes.gohugo.io/) Хьюго.

На данном сайте используется тема Тейл. Для ее&nbsp;установки нужно скачать тему с&nbsp;[Гитхаба](https://github.com/EmielH/tale-hugo) и&nbsp;распаковать в&nbsp;папке ```/themes/tale```. Либо подключить тему в&nbsp;виде&nbsp;[подмодуля](https://git-scm.com/book/ru/v2/Инструменты-Git-Подмодули):
```console
git submodule add https://github.com/EmielH/tale-hugo.git themes/tale
```
В&nbsp;конфиге сайта следует указать тему:
```console
echo "theme = 'tale'" >> config.toml
```
